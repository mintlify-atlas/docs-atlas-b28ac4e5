## Overview

Rushed AI uses a credit-based system to manage AI model usage. Credits are deducted based on the tokens consumed by Claude Sonnet and Haiku models when you interact with the AI.

<Info>
  **Credit Value**: Each credit is worth $0.30 USD
</Info>

## Credit Calculation

Credits are calculated based on token usage with a 3x markup over base API costs:

### Claude Sonnet 4.5

- **Input tokens**: $3 per 1M tokens (base cost)
- **Output tokens**: $15 per 1M tokens (base cost)
- **Final cost**: Base cost × 3, converted to credits

### Claude Haiku 4.5

- **Input tokens**: $1 per 1M tokens (base cost)
- **Output tokens**: $5 per 1M tokens (base cost)
- **Final cost**: Base cost × 3, converted to credits

<Note>
  The 3x markup covers infrastructure, platform features, and ongoing development costs.
</Note>

## Credit Formula

Credits are calculated using this formula:

```typescript
function calculateCredits(modelId: string, inputTokens: number, outputTokens: number): number {
    let inputCost = 0;
    let outputCost = 0;

    if (modelId.includes("sonnet")) {
        // Sonnet 4.5 - $3 per 1M input / $15 per 1M output
        inputCost = 0.000003;
        outputCost = 0.000015;
    } else {
        // Haiku 4.5 - $1 per 1M input / $5 per 1M output 
        inputCost = 0.000001;
        outputCost = 0.000005;
    }

    // Apply 3x Margin
    const totalApiCost = (inputTokens * inputCost) + (outputTokens * outputCost);
    const finalRetailCost = totalApiCost * 3;

    return finalRetailCost / 0.30; // CREDIT_COST_USD
}
```

## Credit Types

### Monthly Credits

- Granted at the start of each billing period
- Based on your subscription plan (50, 150, or 400)
- Do not roll over to the next month
- Consumed first before credit pack credits

### Credit Pack Credits

- Purchased separately as one-time top-ups
- Added immediately to your balance
- Persist across billing cycles
- Consumed after monthly credits are depleted

<Info>
  Your monthly credits are consumed first. Credit pack purchases act as a persistent buffer for heavy usage periods.
</Info>

## Credit Balance Management

Your credit balance includes:

1. **Current Balance**: Total available credits (monthly + pack)
2. **Monthly Credits Used**: Credits consumed from your monthly allocation
3. **Pack Credits Balance**: Remaining credits from purchased packs

### Balance Deduction Order

```typescript
const monthlyRemaining = Math.max(0, creditsMonthly - monthlyCreditsUsed);
const packConsumed = Math.max(0, creditsToDeduct - monthlyRemaining);
const newPackBalance = Math.max(0, currentPackBalance - packConsumed);
```

## Transaction History

All credit transactions are logged with:

- Transaction type (debit, credit, grant, refund)
- Amount changed
- Input/output tokens (for debit transactions)
- Balance before and after
- Timestamp
- Related entity (conversation, project, etc.)

You can view your transaction history in your account dashboard to track credit usage over time.
